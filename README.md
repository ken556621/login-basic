## Installation
1. Github clone

    `git clone https://github.com/ken556621/login-basic.git`
    
2. Use terminal open file 'login-basic'

     `cd login-basic`
     
3. NPM install

    `npm install`
    
4. NPM run start

    `npm run start`
    
5. Open browser type

    `localhost:3000`
